# Creación de un Repositorio Remoto en GitHub y Sincronización

Un repositorio remoto es una copia del proyecto almacenada en GitHub. Permite guardar respaldo del código, compartirlo y entregar tareas. Se conecta con el repositorio local para sincronizar cambios mediante Git.

---

## Crear repositorio remoto en GitHub

1. Entrar a https://github.com
2. Iniciar sesión con la cuenta personal
3. Hacer clic en el botón **New repository**
4. Escribir el nombre del repositorio
5. Elegir si será público o privado
6. NO marcar la opción de crear README si ya existe repositorio local
7. Hacer clic en **Create repository**

---

## Preparar repositorio local

Abrir la terminal dentro de la carpeta del proyecto.
