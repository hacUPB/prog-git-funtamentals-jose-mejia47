# Uso de la Consola — Comandos Básicos

En esta unidad aprendí a utilizar la consola (Git Bash / terminal) para navegar entre carpetas, crear directorios y archivos, y ejecutar comandos de Git. La consola permite trabajar de forma rápida y precisa sobre el sistema de archivos y los repositorios.

## Navegación entre directorios

La navegación se realiza usando rutas y comandos específicos.

Comandos principales:


- `ls` → lista los archivos y carpetas
- `cd carpeta` → entra a una carpeta
- `cd ..` → sube un nivel
- `cd /` → va a la raíz
- `cd ~` → va al directorio del usuario

## Creación de archivos y carpetas

- `mkdir nombre` → crea una carpeta
- `touch archivo.txt` → crea un archivo vacío

## Eliminación

- `rm archivo.txt` → elimina archivo
- `rm -r carpeta` → elimina carpeta

## Buenas prácticas aprendidas

- Verificar siempre la ruta con `pwd`
- Listar contenido con `ls` antes de ejecutar cambios
- Usar nombres claros para carpetas y archivos
- Trabajar dentro de una carpeta de proyectos
